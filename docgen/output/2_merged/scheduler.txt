# Scheduling

## Scheduling

Jac applications support various scheduling mechanisms for background task execution and automation.

### Features

*   **Cron-based Scheduling**: Execute tasks at specified times or intervals using cron expressions.
*   **Interval-based Tasks**: Run tasks repeatedly after a fixed duration.
*   **One-time Date Triggers**: Schedule tasks to run once at a specific future date and time.
*   **Task Automation**: Automate recurring data processing and workflows.
