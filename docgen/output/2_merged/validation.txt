# Validation

## Validation

### Assertions
The `assert` statement is used for validating conditions. If the condition evaluates to `False`, an `AssertionError` is raised.

```jac
assert condition, "Optional error message"
```

**Example:**
```jac
node Person {
    has name: str;
    has age: int;
    init {
        assert self.age >= 0, "Age cannot be negative.";
        assert len(self.name) > 0, "Name cannot be empty.";
    }
}

# Valid
spawn node Person(name="Alice", age=30);

# Invalid - will raise AssertionError
# spawn node Person(name="", age=25);
# spawn node Person(name="Bob", age=-5);
```
